# 🚀 Ola Electric Sales Analysis Dashboard | Power BI

## Project Overview

This project presents an interactive **Sales Analysis Dashboard** developed in **Microsoft Power BI** using a simulated **Ola Electric Sales** dataset. The dashboard provides valuable insights into sales performance, customer behavior, regional trends, and product performance through interactive visualizations and key business metrics.

The objective of this project is to demonstrate the complete Business Intelligence workflow—from data cleaning and transformation to data modeling, DAX calculations, visualization, and business insight generation.

---

## 🎯 Project Objectives

* Analyze overall sales performance.
* Monitor revenue and order trends.
* Compare sales across different scooter models.
* Identify high-performing states and cities.
* Analyze customer demographics and payment preferences.
* Build an interactive dashboard for business decision-making.

## 📂 Data Cleaning

The dataset was cleaned using **Power Query** by:

* Removing duplicate records
* Handling missing values
* Removing blank rows
* Correcting data types
* Trimming and cleaning text
* Standardizing categorical values
* Creating a dedicated Date Table
* Building relationships between tables

---

## 📈 Business Insights

* Premium scooter models generated the highest revenue.
* Revenue is concentrated in major Indian states and cities.
* Company-owned stores performed better than other sales channels.
* Full Payment and EMI are the most preferred payment methods.
* Customer ratings indicate high overall satisfaction.
* Interactive filtering helps analyze trends by date, model, state, and customer segment.
